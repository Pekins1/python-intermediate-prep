# Python Intermediate Certification Prep

This repository contains Python intermediate level practice code and exercises for certification preparation.

## Setup

1. Clone this repository
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. Install dependencies (if any): `pip install -r requirements.txt`

## Usage

Run the main script:
```bash
python main.py
```
